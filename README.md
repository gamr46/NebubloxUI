# NebubloxUI

Universal Script Hub for Roblox — powered by ANUI.

## Quick Start

Execute this in your executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/LilNugOfWisdom/NebubloxUI/main/loader.lua"))()
```

The loader will:
1. Detect which game you're in
2. Auto-load the correct script
3. If undetected, show a manual game selector
